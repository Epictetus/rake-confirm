# Rake::Confirm

 confirmation for destructive Rails task. example for db:drop.

## Installation

Add this line to your application's Gemfile:

    group :production do
      gem 'rake-confirm'
    end

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rake-confirm
